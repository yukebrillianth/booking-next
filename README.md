# booking-next

Simple README untuk project web booking berbasis Next.js.

## Deskripsi

Aplikasi frontend sederhana untuk sistem booking. Tersedia fitur dasar: daftar layanan, pemesanan, dan konfirmasi.

## Prasyarat

- Node.js >= 16
- npm atau yarn

## Instalasi

1. Clone repository:
   git clone <repo-url>
2. Pasang dependensi:
   npm install
   # atau
   yarn

## Menjalankan (development)

npm run dev

# atau

yarn dev
Buka http://localhost:3000

## Build dan production

npm run build
npm start

# atau

yarn build
yarn start

## Variabel lingkungan

Buat file `.env.local` di root dan tambahkan contoh:
NEXT_PUBLIC_API_URL=http://localhost:3000/api
NEXT_PUBLIC_APP_NAME=booking-next

## Struktur singkat

- pages/ — halaman Next.js
- components/ — komponen UI
- public/ — aset statis
- styles/ — CSS / Tailwind
